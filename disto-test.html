Disto X3 Receiver

โปรแกรมง่าย ๆ สำหรับรับข้อมูลจาก Leica Disto X3 บน iPhone
สถานะ: พร้อมรับข้อมูลจาก Disto X3 (Keyboard Mode)

วิธีใช้:
1. เปิด Keyboard Mode / HID บน Disto X3
2. แตะช่องรับค่าด้านล่างเพื่อเลือกช่อง (หรือปล่อยไว้ที่ช่องอัตโนมัติ)
3. ยิงเลเซอร์จาก Disto X3
4. ระบบจะตรวจจับและแสดงค่าที่ได้รับ พร้อมระบุประเภท (ระยะ/มุม/อื่น ๆ)

ข้อมูลที่ได้รับจาก Disto X3

[เริ่มต้น] ระบบพร้อมรับข้อมูลจาก Disto X3

ข้อมูลที่สามารถเรียกดูจาก Disto X3 ได้

Disto X3 สามารถส่งข้อมูลต่อไปนี้ผ่าน Keyboard Mode (ขึ้นกับการตั้งค่าบนเครื่อง):

ระยะตรง ระยะแนวนอน ระยะแนวตั้ง มุมราบ มุมดิ่ง พื้นที่ ปริมาตร ค่าติดลบ หน่วย (ม./ฟุต)

หมายเหตุ: รูปแบบข้อมูลที่ได้รับขึ้นกับโหมดการทำงานของ Disto X3 และการตั้งค่า Output Format ในเครื่อง
